    Техническое задание

Требуется создать клиентскую часть веб-сервиса по созданию
новостных ботов для Telegram. Каждый пользователь может
создавать, редактировать и удалять ботов.

Сервис состоит из следующих страниц:
● страница просмотра списка созданных ботов
● Модальное окно создания нового бота
● Модальное окно редактирования информации о ботах



    Страница просмотра списка созданных ботов

Страница со списком является главной страницей сервиса и кнопкой
создать бота. Каждый элемент списка содержит:
● название бота;
● кнопку удалить.

При нажатии на элемент списка пользователя всплывает модальное
окно с возможностью редактирования информации о боте.

При нажатии на кнопку удалить бот удаляется из списка.

При нажатии на кнопку создать бота всплывает модальное окно с
формой создания бота.



    Модальное окно создания/редактирования ботов

Для модальных окон создания ботов и редактирования использовать
одинаковую форму. Форма состоит из следующих элементов:
● поле для ввода названия,
● поле для ввода описания
● drag and drop поле для загрузки картинки;
● date picker для ввода даты;
● кнопка с сохранением результата;

При создании бота после нажатия на кнопку сохранить модальное
окно закрывается и бот добавляется в конец списка.

Приредактировании после нажатия на кнопку сохранить модальное окно
не закрывается.
